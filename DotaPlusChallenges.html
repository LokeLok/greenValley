Hero Challenges a Never Ending Story


Introduction

Dota 2 is an ever evolving game and the game has changed a lot since Hero Leveling was introduced with Dota Plus back in March of 2018. Since then there hasn't been many changes to how this system work, with a couple of exceptions. We've gotten Grandmaster Tier as maximum level and Hero Relics gives a small amount of experience towards Hero Leveling. Other than that not much has change and that's alright since it's a good system. The current system has one big flaw; Hero Challenges. There are way too many unreasonable Challenges, which has never been addressed by the Dota devs. !!!Challenges that affects enemy Heres should show number going up if enemy is in FoW


This is a final plea to the devs to cleanup and fix every single Hero Challenge. Hero Challenges should be REASONABLE & WORK.

It's time to let go of certain types of Challenges, cleanup the rest. Give them all the same formatting keeping it short, simple, concise and consistent.


Check out the current Challenge and its flaws here. Challenges that should be removed can be found here. All the new suggestions are over here. And lastly here are some UI isssues connected with Hero Challenges.


Challenges Interfering with Gameplay

Challenges should never influence players decision making. Therefore any Challenges that reqiures item's, facet's, talent tree's, game time, ability combo's or enemy depended should be removed.

When it comes to these types of Challenges, it's time to stop with hit 3+ 4+ enemy Heroes with an ability.

Challenges that you need to use abilities in a combo should be removed. why? They need to be done in a specific order, which interfere with gameplay.


Bugged Challenges

Challenges might not work, partially work, have wrong icon or the description and function of the Challenge have different outcome.


Examples of Challenges not working.

  • Deal 2500 / 5000 / 7500 damage to enemy Heroes with Marksmanship. 7+ years and has never worked
  • Deal 3000 / 5000 / 7000 damage to enemy Buildings with Spirit Bear. Stopped working after Spirit Bear became Universial
  • Get 4 / 7 / 10 assists or kills by friendly Heroes buffed with Warcry. Doesn't count kills
  • Prevent 3000 / 5000 / 7000 Physical damage on yourself and friendly Heroes by using Warcry. Counts all damage block on the map

Examples of Challenges partially working

  • Deal 4000 / 6000 / 8000 to enemy Heroes with Nether Ward. Doesn't work with Siphoning Facet
  • Multicast enemy Heroes a total of 10 / 20 / 30 times. Doesn't work with Fat Chance Facet

Examples of Challenges where description and function are different

  • Deal 6000 / 12000 / 18000 damage to enemy Heroes with Arc Lightning. Counts damage done to creeps
  • Deward 1 / 2 / 3 or more enemy Observer Wards. Counts allied observer wards
  • Kill 6 / 12 / 18 enemy Heroes that are slowed by Reflection. Counts assists
  • Kill an enemy hero in 15 seconds after teleporting in 1 / 2 / 4 times. Counts assists and it's a 20 seconds timeframe

Outdated Challenges

This will always be an issue, but by removing, simplify and standardize Challenges, it will become easier for the community to give feedback on what's wrong.


Examples of Challenges being outdated

  • Deal 3000 / 6000 / 9000 damage to enemy Heroes with Searing Arrows.
  • Deal 4000 / 6000 / 8000 mana burn to enemy heroes with Mana Burn.
  • Deal 4000 / 8000 / 12000 damage to enemy Heroes with Poison Nova that is upgraded with Aghanim's Scepter.
  • Deal 4000 / 8000 / 12000 damage to enemy Heroes with the Second Chakram.

Unattainable Challenges

There are many Challenges that are unattainable beacuse of the numbers are unreasonable. I don't think this is a job for the community to keep track of and give feedback, since we've no way to sandbox test this ourselves. Dota devs are the ones that can track each Challenge and their completion rate. Gather data and adjust the numbers accordingly and tune them to match an average Dota game. Three star Challenges should be the most likely outcome in an average Dota game.


Template Used for Writing Hero Specific Challenges

I've used already exsisting formatting from the devs and applied the best and shortes way to explain a Challenge. I've tweak a few to match up with the rest. This is my personally recommendation based on how the Challenges are currently written, together with the most common Dota terms. If you think there is a better way to write some of them, please keep it short, concise and the same across all Heroes.


  • Accumulate x / x / x [attribute/damage] with [ability].
  • Buff allied Heroes with [ability] for x / x / x seconds. Count self towards the Challenge
  • Buff yourself with [ability] for x / x / x seconds.
  • Burn x / x / x mana from enemy Heroes with [ability].
  • Create x / x / x illusions with [ability].
  • Deal x / x / x damage to enemy buildings while you are affected by [ability]. Includes all type of buildings
  • Deal x / x / x damage to enemy buildings with [ability]. Includes all type of buildings
  • Deal x / x / x damage to enemy Heroes that are affected by [ability].
  • Deal x / x / x damage to enemy Heroes while you are affected by [ability]. Count any kind of damage done by you from any source
  • Deal x / x / x damage to enemy Heroes with [ability].
  • Debuff enemy Heroes with [ability] for x / x / x seconds.
  • Disjoint enemy abilities with [ability] x / x / x times.
  • Dispel debuffs from allied Heroes with [ability] x / x / x times. Count self towards the Challenge
  • Dispel debuffs with [ability] x / x / x times.
  • Evade x / x / x damage with [ability]. Count damage done from anything
  • Get x / x / x kills or assists on enemy Heroes that are affected by [ability]. Counts if the enemy dies from the ability itself
  • Get x / x / x kills or assists on enemy Heroes while you are affected by [ability].
  • Heal x / x / x health to allied Heroes with [ability]. Count self towards the Challenge
  • Heal x / x / x health with [ability]. If the Challenge needs an enemy to heal from e.g. lifesteal, make it count heal from any units
  • Hit enemy Heroes with [ability] x / x / x times.
  • Kill x / x / x enemy creeps with [ability]. Includes lane creeps, neutrals and summons
  • Kill x / x / x enemy Heroes with [ability]. This should only be used on abilities that are guaranteed or likely to get kills
  • Prevent x / x / x damage to allied Heroes with [ability].
  • Prevent x / x / x damage with [ability].
  • Restore x / x / x mana to allied Heroes with [ability]. Count self towards the Challenge
  • Restore x / x / x mana with [ability].
  • Summon x / x / x [summons] with [ability].
