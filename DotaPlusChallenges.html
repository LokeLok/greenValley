Everything Wrong Hero Challenges


Introduction

Dota 2 is an ever evolving game and a lot has change since Hero Leveling was introduced with Dota Plus back in March of 2018. Since then there hasn't been many changes to how this system work, with the exceptions of two things. We've gone from Master Tier(level 25) being the max, to Grandmaster(level 30) and Relics gives a small amount of experience towards Hero Leveling. Other than that, not much has change and that's alright, since it's a fairly good system.


Hero Challenges being the main part of gaing experience has been a good thing. There is just one big flaw, they SUCK.


A lot of Challenges has been removed, but many times more new have been suggested.



There are also some other minor UI issues connected with Hero Challenges.


Challenges Interfering with Gameplay

Challenges should never influence players decision making. Therefore any Challenges that reqiures Item's, Facet's, Talent Tree's, Game Time or the enemies for you to complete Challenges, should be removed. If you really want to keep item Challenges let players selecet which item Challenge to chose.


Bugged Challenges

Common bugs: not working, partially working, description and function have different outcome.


Examples of Challenges that don't work.

  • Deal 2500 / 5000 / 7500 damage to enemy Heroes with Marksmanship. | Has never worked
  • Deal 3000 / 5000 / 7000 damage to enemy Buildings with Spirit Bear. | Stopped working after Spirit Bear became Universial

Exampals of Challenges that partially work

  • Multicast enemy Heroes a total of 10 / 20 / 30 times. | Doesn't work with Fat Chance Facet

Examples of Challenges that work different from their description

  • Deal 6000 / 12000 / 18000 damage to enemy Heroes with Arc Lightning. | Counts damage done to creeps(possible to any unit).
  • Kill 6 / 12 / 18 enemy Heroes that are slowed by Reflection. | Counts assists.

It's important that the function of the Challenge works as described.


Combo Challenges

Challenges that has the requierment in them should be removed.


Outdated Challenges

This will always be an issue, but removing some Challenges and by standardize the rest. It will improve readablity, tracking bugs and mainting them after gameplay patches much easier.

  • Deal 4000 / 8000 / 12000 damage to enemy Heroes with Poison Nova that is upgraded with Aghanim's Scepter.
  • Deal 3000 / 6000 / 9000 damage to enemy Heroes with Searing Arrows.
  • Deal 4000 / 8000 / 12000 damage to enemy Heroes with the Second Chakram.

Unattainable Challenges

This is a major issues, the share amount of Challenges that are unattainable beacuse of the numbers are unreasonable is not ok. It's time for the dev team to start track every single Challenge completion and adjust the numbers deemed too high or low.


Description a Jungle of 10.000 Spiches

Challenges that wanna express the same thing, has so many different ways to do so. There are no reason for this.


Challenges that says 'to allied Heroes' should ALWAYS include yourself.


Exampels of Challenges that describe the same thing

  • Simultaneously stun two or more enemy heroes with one Unstable Concoction, 3 / 5 / 7 times.
  • Land a Blood Rite that silences 2 or more enemy Heroes, 3 / 5 / 7 times.
  • Cast a Shuriken Toss that bounces on at least 2 enemy heroes 4 / 8 / 12 times.
  • Hit 2 or more enemy Heroes with Silence a total of 5 / 7 / 9 times.
  • Damage 2 or more enemy Heroes with a single Splinter Blast, 3 / 6 / 9 times.
  • Bind 2 enemy Heroes with Shackleshot 1 / 2 / 3 times.
  • Hit two or more enemy heroes with Pulse a total of 6 / 12 / 18 times.

All these Challenges can be written like this: Hit multiple enemy Heroes with [ability] x / x / x times.

  • Hit multiple enemy Heroes with Unstable Concoction 3 / 5 / 7 times.
  • Hit multiple enemy Heroes with Blood Rite 3 / 5 / 7 times.
  • Hit multiple enemy Heroes with Shuriken Toss 4 / 8 / 12 times.
  • Hit multiple enemy Heroes with Silence 5 / 7 / 9 times.
  • Hit multiple enemy Heroes with Splinter Blast 3 / 6 / 9 times.
  • Hit multiple enemy Heroes with Shackleshot 1 / 2 / 3 times.
  • Hit multiple enemy Heroes with Resonant Pulse 6 / 12 / 18 times.

Short, simple and easy to maintain.


Template for Writing Challenges for Heroes

  • Buff allied Heroes with [ability] for x / x / x seconds.
  • Buff yourself with [ability] for x / x / x seconds.
  • Catch multiple enemy Heroes with [ability] x / x / x time. | This should only be used if 'hit multiple' doesn't make sense
  • Deal x / x / x damage to enemy buildings with [ability].
  • Deal x / x / x damage to enemy Heroes with [ability].
  • Deal x / x / x damage to enemy Heroes that are affected by [ability].
  • Deal x / x / x damage to enemy Heroes while you are affected by [ability]. | This should always count any damage done by you
  • Debuff enemy Heroes with [ability] for x / x / x seconds.
  • Dispel debuffs from allied Heroes with [ability] x / x / x times.
  • Dispel debuffs with [ability] x / x / x times.
  • Get x / x / x kills or assists on enemy Heroes that are affected by [ability]. | Make sure it always counts if you get the kill with said ability
  • Get x / x / x kills or assists on enemy Heroes that are inside [ability].
  • Get x / x / x kills or assists on enemy Heroes while you are affected by [ability].
  • Heal x / x / x health to allied Heroes with [ability].
  • Heal x / x / x health with [ability]. | Make this Challenge heal from any units (not only Heroes)
  • Hit enemy Heroes with [ability] x / x / x times.
  • Hit multiple enemy Heroes with [ability] x / x / x times.
  • Kill x / x / x enemy creeps with [ability]. | This always includes lane creeps, neutrals and summons
  • Kill x / x / x enemy Heroes with [ability]. | This should only be used on abilities that are likely to get kills
  • Prvent x / x / x damage from enemy Heroes with [ability].
  • Restore x / x / x mana to allied Heroes with [ability]. | Make any type of mana given work
  • Restore x / x / x mana with [ability]. | Make any type of mana given work
